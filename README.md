# Prompt Playground

A React application for writing, testing, comparing, and versioning AI prompts. It provides a simple workspace to experiment with prompts, track improvements, and manage prompt versions in one place.

## Live Demo

https://prompt-playground-eta.vercel.app

## Features

- Write and test AI prompts
- Organize prompts in one place
- Save prompt versions with change notes
- Track prompt iterations
- Clean and responsive user interface
- Fast development with Vite
- Deployed on Vercel

## Tech Stack

- React
- JavaScript
- Vite
- CSS
- Vercel
